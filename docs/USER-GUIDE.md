## Creating Invitations

### Step 1: Basic Information

1. Click **Create Invitation**
2. Enter couple names:
   - **Bride/Partner 1 Name**
   - **Groom/Partner 2 Name**

### Step 2: Wedding Details

1. Set **Wedding Date** and **Time**
2. Set **Reception Date** and **Time** (if different)
3. Enter **Venue Name**
4. Enter **Venue Address**
5. Paste **Google Maps Link**

### Step 3: Couple Information

For each partner, provide:
- **Name**
- **Subtitle** (e.g., "The Bride")
- **Bio** (200 characters)
- **Photo** (drag & drop or click)
- **Parents' Names**

### Step 4: Media

1. **Upload Gallery Images:**
   - Drag and drop multiple images
   - Or click to browse
   - Maximum 5MB per image
   - Recommended: High-quality photos

2. **Cover Image:**
   - Main image for invitation
   - Recommended size: 1200x800px

3. **Background Music:**
   - Upload MP3 or WAV file
   - Maximum 20MB
   - Will play on invitation

### Step 5: Wedding Story

1. Click **Generate with AI** for suggestions
2. Or write your own story (200-300 words)
3. Preview before saving

### Step 6: Guests

1. **Enter guest limit** (leave blank for unlimited)
2. Add **bridesmaids** and **groomsmen**
3. Add **family members** with relationships

### Step 7: Events

1. Add event schedule:
   - **Ceremony time**
   - **Photos**
   - **Cocktail hour**
   - **Reception**
   - etc.

2. Use **AI Generator** for suggestions
3. Edit times as needed

### Step 8: Theme & Design

1. Select **Theme:**
   - Royal Gold
   - Minimal White
   - Rose Pink
   - Emerald Green
   - Classic Black
   - Luxury Red
   - Ocean Blue
   - Custom

2. Choose **Color Palette:**
   - Use suggested colors
   - Or customize
   - Click to adjust

3. Add **Custom Logo** (optional)

### Step 9: RSVP Settings

1. Check **Enable RSVP** to allow responses
2. Set guest limit (0 = unlimited)
3. Choose **meal preferences** options
4. Set RSVP deadline

### Step 10: SEO & Social

1. Enter **SEO Title** (60 characters)
2. Enter **SEO Description** (160 characters)
3. Upload **Social Share Image** (1200x630px)
4. Use **AI Generator** for suggestions

### Step 11: Preview & Publish

1. Click **Preview** to see invitation
2. Make adjustments if needed
3. Click **Publish** to make live
4. Copy share link and QR code

## FAQ

### General Questions

**Q: How many invitations can I create?**
A: Unlimited! Create as many as you need.

**Q: Can I edit an invitation after publishing?**
A: Yes, click Edit and your changes take effect immediately.

**Q: Can guests edit their RSVP?**
A: Yes, they can update their response anytime.

### Technical Questions

**Q: What image formats are supported?**
A: JPG, PNG, GIF, WebP (up to 5MB each)

**Q: Can I use custom domain?**
A: Yes, during setup or in settings.

**Q: Is my data secure?**
A: Yes, all data is encrypted and secure.

### Troubleshooting

**Q: Images won't upload**
A: Check file size (<5MB) and format (JPG/PNG)

**Q: AI features not working**
A: Verify OpenAI API key in settings

**Q: RSVP button not appearing**
A: Ensure RSVP is enabled in invitation settings
